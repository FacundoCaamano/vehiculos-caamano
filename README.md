# vehiculos-caamano

Hola, mi nombre es Facundo Caamaño  tengo 21 años y este es mi proyecto para mi cursada.
Se trata de una página responsiva de una concesionaria de autos usados, es ficticia es decir que dichos autos y concesionarios no existen. Para el proyecto 
use html, css, scss (con la extensión de VSC), git, github, bootstrap y vercel. Mi pagina cuenta con 5 html (inicio, vehiculos, Financiacion, Concesionario, contacto) 
que son navegables y 6 html que se encuentran dentro de financiación (una para cada modelo de auto). Utilice bootstrap para las cartas, el nav y el carrousel,
con Scss cree estilos.scss y ahí importe todos los parciales. Use git para versionar el codigo, tambien cree una rama1 donde la usaba para trabajar, 
si el código funcionaba bien, lo mergeaba a la rama master y de ahí lo subí a github. Elegí vercel para alojar mi pagina por la comodidad de que usa los archivos
del repositorio de github. El footer cuenta con el logo de twitter y facebook que son clickeables y te abren una ventana nueva con dicha red social. Para el
el proyecto utilice las prácticas enseñadas, como usar el ampersand, camelCase, tabulación, etiquetas semánticas, metodología beem etc. Aplique una mejora al SEO
optimizando las imágenes en Tinyjpg, respetando el uso de h1 h2 h3 y meta datos de keywords y content.
